# Neuron Kanban — продуктовое описание (RU)

## Назначение

Веб-приложение для управления задачами команды в стиле **Канбан Нейрона**: доска колонок, диаграмма Ганта, ИИ-помощник (разбор текста в задачи и чат по контексту проекта), интеграция с Telegram-ботом.

## Пользовательские сценарии

### Доска

- Задачи распределены по колонкам жизненного цикла: глобальный бэклог (верхняя узкая полоса), бэклог спринта, продуктовая и техническая разработка, тестирование, выполнено в спринте; внизу — **«Выполнено давно»** (горизонтальный архив).
- Перетаскивание за **ручку** слева на карточке; клик по телу карточки открывает редактирование.
- **История** — кнопка «?»; при открытой истории перетаскивание не начинается с области просмотра (ручка отделена).
- **Фильтры** (исполнитель, блок Нейрона, приоритет): несовпадающие карточки отображаются бледнее; порядок и DnD сохраняются для всех задач.
- **Удаление** задач — только у администраторов (в модалке редактирования). С карточки удаление убрано.

### Поля задачи

- Название, описание, колонка, исполнитель, приоритет, сложность, даты, зависимости.
- **Блок Нейрона** — к какой части платформы относится задача: чат, агентская система, функции, документы; сервисы: speechwriting, генератор поручений, правовой контроль, анализ материалов процессов, Rejus-strip.
- **Артефакты** — вложения (имя, MIME, data URL), до 3 файлов по 400 КБ в интерфейсе.

### Гант

- Неделя / день / месяц; таблица под диаграммой показывает сроки и зависимости.
- При **перетаскивании** полосы даты в таблице обновляются в реальном времени (превью).
- После отпускания мыши — **предупреждение** о намеренном сдвиге сроков; можно отметить «не показывать 5 минут» (хранится в `sessionStorage`).
- Выгрузка **CSV с BOM** для открытия в Excel.
- Визуально спокойная однотонная палитра полос и строк сетки; в режиме «День» убраны выраженные «зебры» строк.

### ИИ

- **Задачи на доску** — текст отправляется в разбор; результат — предпросмотр пакета карточек (без отображения P/D в упрощённой форме), подтверждение создаёт задачи.
- **Общение** — вопрос к модели с контекстом текущих задач; **карточки не создаются** автоматически.

### Локализация

- Переключатель **Русский / English** в шапке; сохранение в `localStorage`.

## Роли

- **Пользователь** — редактирование задач, смена колонок (кроме дат без прав админа — по правилам API).
- **Администратор** — смена дат у существующих задач, удаление задач.

## Ценность

Единая точка для бэклога, спринта и архива; связь задач с подсистемами Нейрона; быстрый ввод через ИИ; наглядные сроки в Ганте с контролем случайных сдвигов.
